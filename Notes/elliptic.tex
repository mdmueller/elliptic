  There is a map $\pi$ from this moduli stack to $\overline{\mathcal M}_{1,1}$, and the image of the virtual fundamental class is an element of $A^0(\overline{\mathcal M}_{1,1})\cong \Z$.
  What is this number?
\end{prob}
